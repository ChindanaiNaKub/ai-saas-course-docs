Lesson 0019 · Repeatable sales pipeline

สร้าง pipeline ที่พาลูกค้าใหม่เข้ามาเรื่อยๆ

บทเรียนนี้สอนทักษะเดียว: เปลี่ยน package ที่ขายได้ให้กลายเป็นระบบ pipeline รายสัปดาห์ พร้อมตัวเลข leads, replies, calls, proposals, และ closes ที่คุณคุมได้.

กับดัก

มี package แล้ว แต่หาลูกค้าเป็นรอบๆ ตามอารมณ์: โพสต์บ้าง ทักบ้าง หายบ้าง แล้วสรุปว่าตลาดไม่ซื้อ ทั้งที่จริง pipeline ยังไม่สม่ำเสมอ.

เป้าหมาย

มี one channel, one ICP list, one offer, one weekly cadence และ scoreboard ที่บอกว่าปัญหาอยู่ตรงไหนใน funnel.

1. Pipeline คือระบบ ไม่ใช่ความหวัง

บทเรียนแรกๆ เราคุยว่า distribution สำคัญกว่าแค่สร้าง product. ตอนนี้คุณมี wedge และ package แล้ว จึงต้องสร้างระบบที่พาคนเหมาะๆ เข้ามาคุยอย่างต่อเนื่อง. YC และ First Round ย้ำเรื่อง founder-led sales เพราะช่วงแรก founder ต้องเรียนรู้จากตลาดโดยตรง ไม่ใช่ outsource ความเข้าใจลูกค้าเร็วเกินไป.

ถ้าคุณไม่มีจำนวน outreach, replies, calls, proposals, และ closes ต่อสัปดาห์ คุณยังไม่มี pipeline มีแค่กิจกรรมขายกระจัดกระจาย.

2. Pipeline 5 ขั้น

StageคำถามMetric
Lead list เรารู้จัก buyer ที่ตรง ICP กี่ราย qualified leads added/week
Outreach เราทักคนตรง segment กี่ราย messages sent/week
Reply message เจาะ pain พอให้ตอบไหม reply rate
Discovery call คนที่ตอบมี pain, budget, urgency ไหม qualified calls/week
Close offer ชัดพอให้จ่ายหรือ commit ไหม paid pilots or subscriptions/month

3. หา bottleneck ให้ถูก

Lead problem

หา lead ไม่พอหรือ lead ไม่ตรง ICP. แก้ด้วย list source, trigger, community, directory, partner, referral.

Message problem

ทักเยอะแต่ไม่มีคนตอบ. แก้ด้วย pain ที่ specific, proof, short ask, better targeting.

Offer problem

มี call แต่ไม่ close. แก้ด้วย qualification, package, price, urgency, risk reversal, next step.

4. Weekly cadence ที่ founder ทำได้จริง

อย่าเริ่มจาก CRM ใหญ่โต. เริ่มจาก spreadsheet หรือ notes ก็พอ แต่ต้องมี rhythm.

ทุกสัปดาห์

  • เพิ่ม qualified leads 25-50 ราย
  • ส่ง outreach 30-80 ข้อความ
  • follow up คนที่เงียบ 1-2 รอบ
  • ทำ discovery calls 3-8 ครั้ง
  • ส่ง proposal เฉพาะคน qualified

อย่าทำแบบนี้

  • ทักคนไม่ตรง ICP เพื่อให้ตัวเลขดูเยอะ
  • เปลี่ยน channel ทุก 2 วัน
  • ไม่ follow up แล้วคิดว่าเขาไม่สนใจ
  • ขาย package ให้คนที่ไม่มี pain/budget
  • ไม่บันทึกเหตุผลที่แพ้ดีล

5. แบบฝึกหัด: Pipeline math

ใส่เป้าหมายและ conversion rate เพื่อคำนวณ activity ต่อเดือน. ตัวคำนวณนี้เป็น JavaScript ในไฟล์ ไม่เรียก AI และไม่ส่งข้อมูลออกไปไหน.

รอตัวเลข pipeline อยู่.

7. Mini Quiz

สถานการณ์: คุณส่ง outreach 120 ข้อความในเดือนนี้ ได้ reply 3 คน มี call 1 ครั้ง และไม่ close. จุดแรกที่ควรแก้คืออะไร?

เลือกคำตอบหนึ่งข้อ.
