Repeatable Sales Pipeline Cheat Sheet

ใช้หลังคุณมี package ที่พอขายได้. เป้าหมายคือสร้างระบบหาลูกค้าใหม่รายสัปดาห์ และรู้ว่า bottleneck อยู่ที่ lead, message, call, offer, หรือ close.

แก่นคิด

Pipeline ไม่ใช่ความหวัง. Pipeline คือจำนวน qualified leads, outreach, replies, calls, proposals, และ closes ที่วัดได้ทุกสัปดาห์.

5-stage pipeline

StageMetricBottleneck เมื่อ
Lead list qualified leads added/week หา buyer ที่ตรง ICP ไม่พอ
Outreach messages sent/week activity ไม่สม่ำเสมอ
Reply reply rate ICP/message/pain ไม่คม
Discovery call qualified calls/week คนตอบไม่ใช่ buyer หรือ ask ไม่ชัด
Close paid pilots/subscriptions/month offer, price, trust, urgency ยังไม่พอ

Pipeline math

customers target -> proposals -> qualified calls -> replies -> outreach

คำนวณย้อนกลับเสมอ. ถ้าอยากได้ 2 ลูกค้า/เดือน และ close rate 35%, คุณต้องมีประมาณ 6 proposals. ถ้า call-to-proposal 50%, ต้องมี 12 qualified calls. ถ้า reply-to-call 35% และ reply rate 12%, ต้องส่ง outreach ประมาณ 286 ข้อความ/เดือน หรือ 72 ข้อความ/สัปดาห์.

Weekly rhythm

  1. Monday: build lead list และเลือก message angle
  2. Tuesday-Wednesday: send outreach และ follow up
  3. Thursday: run discovery calls และส่ง proposal เฉพาะ qualified buyers
  4. Friday: review funnel metrics, objections, lost reasons, next experiment

Bottleneck map

Lead problem

หา leads ไม่พอหรือไม่ตรง ICP. แก้ list source และ qualification.

Message problem

reply ต่ำ. แก้ pain specificity, proof, short ask, personalization.

Offer problem

call เยอะแต่ไม่ close. แก้ package, pricing, urgency, risk reversal.
