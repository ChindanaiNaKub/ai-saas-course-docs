Channel Experiment Cheat Sheet

ใช้ก่อนเริ่มทดสอบ channel ใหม่. เป้าหมายคือทดสอบอย่างมี sample, metric, และ decision rule ไม่ใช่ลองมั่วแล้วเดาจากความรู้สึก.

แก่นคิด

Channel ต้อง match buyer behavior: เขาอยู่ไหน, เห็น pain เมื่อไหร่, เชื่อใคร, และซื้อผ่านอะไร.

Test card

Fieldต้องเขียน
Channelช่องทางเดียวที่จะ test รอบนี้
ICP sourceจะหา buyer/audience จากไหน
Hypothesisทำไมช่องนี้น่าจะเข้าถึง buyer ได้
Message anglepain + proof + offer
Proof assetmini case, ROI proof, trust note, pilot plan
Activity targetจำนวน outreach/posts/asks/conversations
Pass metricreply, qualified calls, paid pilots, conversion
Decision dateวันตัดสิน keep / iterate / kill

Decision rule

  • Keep: pass metric ถึง และ conversations ตรง ICP
  • Iterate: มี qualified conversations แต่ conversion อ่อน
  • Kill: ทำ activity target ครบแล้วยังไม่มี qualified signal

Minimum samples

  • Cold outbound: 100-200 targeted messages
  • Community: 8-12 high-quality contributions
  • Referral: 20-30 specific asks
  • Partner: 10-20 partner conversations
