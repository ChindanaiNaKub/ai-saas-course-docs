Message Stack Cheat Sheet

Message ที่ดีตอบเร็วว่า ทำเพื่อใคร, pain อะไร, outcome อะไร, proof คืออะไร, และให้ทำอะไรต่อ.

5 Layers

LayerTemplate
ICPสำหรับ [specific buyer]
Painที่เสีย [cost] เพราะ [problem]
Outcomeเราให้ [repeatable output] เพื่อ [action]
Proofเช่น [pilot result / quote / before-after]
CTAเริ่มด้วย [low-risk next step]

Red Flags

  • พูดว่า AI platform แต่ไม่บอกงานที่ช่วย
  • ไม่มี ICP
  • ไม่มี proof
  • CTA ใหญ่เกินไป เช่น "Book enterprise demo" ก่อนมี trust
